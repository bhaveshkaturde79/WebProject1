🦠 WannaCry Ransomware Attack

One of the most significant cyberattacks in history

📌 What Was WannaCry?

WannaCry was a massive ransomware attack that spread worldwide in May 2017 using malware that encrypted files on Windows computers and demanded ransom payments in Bitcoin. It spread rapidly by exploiting a vulnerability in Microsoft Windows called “EternalBlue”, which had been leaked from the U.S. National Security Agency. :contentReference[oaicite:0]{index=0}

🌍 How Did It Spread?

WannaCry used a worm‑like mechanism that allowed it to jump between computers connected on the same network without user interaction. It targeted the Windows Server Message Block (SMB) protocol vulnerability, which was patched by Microsoft — but many systems hadn’t installed the security update yet. :contentReference[oaicite:2]{index=2}

⚠️ Global Impact

WannaCry affected an estimated **300,000+ computers in over 150 countries**, causing widespread disruption across healthcare, government, businesses, and education sectors. :contentReference[oaicite:3]{index=3} One of the most publicised victims was the **National Health Service (NHS)** in the UK, where many systems were infected, leading to appointment cancellations and operational delays. :contentReference[oaicite:4]{index=4} Estimated economic damage from disruption and recovery is in the **billions of dollars**. :contentReference[oaicite:5]{index=5}

🛡️ Why This Attack Mattered

  • ✅ It showed how quickly ransomware can spread across networks and borders.
  • ✅ It exposed risks from unpatched systems still in use.
  • ✅ It focused global attention on cybersecurity preparedness and patch management.
  • ✅ Many organizations lacked up‑to‑date backups, increasing ransom pressure.

🛠️ How to Protect Against Ransomware

  • ✔ Keep software and operating systems fully updated with the latest patches. :contentReference[oaicite:6]{index=6}
  • ✔ Back up important files regularly and keep backups offline or in secure cloud storage. :contentReference[oaicite:7]{index=7}
  • ✔ Use reputable security software with real‑time protection. :contentReference[oaicite:8]{index=8}
  • ✔ Be cautious with email attachments and links from unknown senders. :contentReference[oaicite:9]{index=9}
  • ✔ Disable unneeded services such as outdated SMB protocols. :contentReference[oaicite:10]{index=10}
